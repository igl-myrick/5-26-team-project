# May 2026 Team Project

#### By India Lyon-Myrick, MakeyeB, Josh Butcher

## Description



#### Outline

* _Project will use either the [Colormind](http://colormind.io/) (5 colors) or [Huemint](huemint.com) (4 colors) API._

* _Fully styled site with customizable colors_
* _New palette can be generated for the whole page or for individual sections of the page?_
* _Dropdown of potential colors or allow user to input one or two starting colors?_
* _Color ID, to show what the color is for use in other work?_

#### MVP

* _Successful API call_
* _Results of the API call are displayed on the page somehow_

#### Stretch Goals

* _Handling for problematic color schemes (in case results give poor readability or colors too similar, etc.), could have a fallback color or formula for determining fallback colors_
* _More advanced interpretation of the color schemes returned by API? (displayed in graph or something along those lines)_
* _A way (via array or other means), to save the last few colors "remembered" for use. Maybe clicking the items in this UI area can display color ID for users to call back upon?_
